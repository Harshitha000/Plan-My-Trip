# Plan-My-Trip
Plan My Trip is a trip planning platform that assists users in searching and selecting tourist sites, generating an optimal path between them.The application features a search bar where you can search for any city in India. It utilizes the Geoapify autocomplete API to provide place suggestions while typing. Users can view different tourist sites in a searched location, with images and details for selection. We've utilized the Geoapify Places API and Pixbay API for this information. Using coordinates, we calculate the distances between tourist sites. We then employ the Traveling Salesman Problem (TSP) algorithm to generate the shortest possible route between them.
